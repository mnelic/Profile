About

My name is Mnelisi Mbonani, and I’m a web developer with a diverse set of skills and experience. Currently, I work as a Full Stack Software Developer at Harambee, where I’ve been an integral part of developing and enhancing the SAYouth Partner Portal.


The SAYouth Partner Portal (partners.sayouth.org.za) is a platform designed to help employers connect with young talent. This platform serves as a bridge between employers and approximately 4 million unemployed youth profiles. SAYouth was recently honored as the national pathway manager for unemployed youth by the president.


My primary responsibilities include:

  • Building and optimizing database systems,ncluding stored procedures, views, and functions.
  • Developing the front-end, utilizing Razor views and JavaScript to ensure responsiveness and an intuitive user experience.

Notable Contributions Include:

  • Adding a download button with functionality to export a list of applicants.
  • Implementing role-based permissions for partners to reject application statuses.
  • Enhancing visuals and user experience across the platform.
  • Participating in code reviews on GitHub.
  • Contributing to refinement sessions, analyzing tickets, and providing development estimates.

Before my current role, I completed a Software Developer Internship at Digilink, where I co-developed a File Management System for securely storing hospital records in the cloud. This project sharpened my skills in database design, API development, and client-side implementation, while also giving me valuable experience working in an Agile environment using tools like GitHub and Jira
